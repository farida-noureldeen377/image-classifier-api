# Image Classifier API

A small web service that tells you what's in a picture.

## What it does

You send the API an image, and it sends back a list of the objects it thinks
are in that image, along with how confident it is about each one. For example,
upload a photo of a dog on a beach and you might get back something like
`golden retriever (0.92)`.

Under the hood it uses a pre-trained deep learning model (from PyTorch /
torchvision) to do the actual recognition, wrapped in a [FastAPI](https://fastapi.tiangolo.com/)
HTTP service so any application can use it over a simple web request.

## Requirements

- Python 3.10+
- (Optional) Docker, if you'd rather run it in a container

## Setup

```bash
# 1. Clone the repo and move into the project
cd image-classifier-api

# 2. Create and activate a virtual environment
python -m venv .venv
source .venv/bin/activate        # on Windows: .venv\Scripts\activate

# 3. Install the dependencies
pip install -r requirements.txt

# 4. Create your local config from the example
cp .env.example .env
```

The `.env` file controls a couple of settings:

| Variable                     | Default | What it does                                        |
| ---------------------------- | ------- | --------------------------------------------------- |
| `PORT`                       | `8000`  | The port the API listens on.                        |
| `MODEL_CONFIDENCE_THRESHOLD` | `0.5`   | Hide predictions less confident than this (0–1).    |

## Running it

Locally with uvicorn:

```bash
uvicorn src.api:app --reload --port 8000
```

Once it's running, open http://localhost:8000/docs to see the interactive API
documentation (auto-generated by FastAPI).

### With Docker

```bash
docker build -t image-classifier-api .
docker run -p 8000:8000 --env-file .env image-classifier-api
```

## Using the API

Send an image file to the prediction endpoint:

```bash
curl -X POST http://localhost:8000/predict \
  -F "file=@my_photo.jpg"
```

Example response:

```json
{
  "predictions": [
    { "label": "golden retriever", "confidence": 0.92 },
    { "label": "tennis ball",       "confidence": 0.61 }
  ]
}
```
